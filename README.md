## 📱 Описание

**MyKaspi** — это мобильное Android приложение, разработанное для работы с платформой Kaspi.kz. Приложение предоставляет удобный интерфейс для управления платежами, покупками и другими финансовыми операциями.

## 🚀 Особенности

- 📲 Полнофункциональное мобильное приложение для Android
- 💳 Поддержка работы с Kaspi платежной системой
- 🎨 Современный и интуитивный пользовательский интерфейс
- ⚡ Быстрая и отзывчивая работа
- 🔒 Безопасность данных пользователя

## 📥 Установка

### Требования

- Android версия 5.0 и выше
- Минимум 50 МБ свободного места на устройстве

### Способы установки

#### 1. Прямая установка APK файла

1. Скачайте файл `MyKaspi.apk` из репозитория
2. На вашем Android устройстве перейдите в **Настройки** → **Приложения**
3. Включите опцию **"Неизвестные источники"** (если требуется)
4. Откройте загруженный файл `MyKaspi.apk`
5. Нажмите **"Установить"**

#### 2. Через компьютер (ADB)

```bash
# Подключите устройство через USB
adb devices

# Установите приложение
adb install MyKaspi.apk
```

## 📖 Использование

### Первый запуск

1. Откройте приложение MyKaspi
2. Зарегистрируйтесь или авторизуйтесь с вашим аккаунтом Kaspi
3. Подтвердите номер телефона
4. Готово! Вы можете начать использовать приложение

### Основные функции

- **Платежи** — отправляйте деньги другим пользователям
- **Покупки** — безопасные онлайн-транзакции
- **История операций** — просмотр всех ваших переводов
- **Баланс счета** — контроль вашего финансового положения
- **Настройки** — персонализация приложения

## 🐛 Известные проблемы и решения

### Приложение не устанавливается

- Убедитесь, что у вас достаточно свободного места на устройстве
- Проверьте, включена ли опция установки приложений из неизвестных источников
- Попробуйте удалить предыдущую версию приложения

### Проблемы с авторизацией

- Проверьте подключение к интернету
- Убедитесь, что вы используете правильные учетные данные Kaspi
- Попробуйте очистить кэш приложения в настройках

### Приложение работает медленно

- Закройте ненужные приложения в фоне
- Очистите кэш и внутреннюю память устройства
- Обновите приложение до последней версии
