# MyKaspi v0.1.0 — Первый релиз

## 🚀 Основной функционал:
- **Аутентификация** — вход по номеру телефона с получением кода
- **Просмотр баланса** — отображение текущего баланса счёта
- **Переводы** — отправка денег другим пользователям
- **Оплата услуг** — оплата интернета и других услуг
- **История операций** — просмотр последних транзакций
- **График расходов** — визуализация трат по времени
- **Смена темы** — поддержка тёмной темы
- **Интерфейс** — современный UI с фиолетовыми акцентами
